# Clinical Dosage and Safety Analyzer 

## Overview
A Python-based tool designed to assist healthcare professionals in calculating safe drug dosages based on patient weight and identifying potential drug-drug interactions.

## Key Features
- **Dosage Calculation:** Automatically calculates doses (mg) based on weight (kg).
- **Safety Limits:** Checks if the calculated dose exceeds the maximum daily limit.
- **Interaction Checker:** Alerts users to known interactions between medications.
- **Data Visualization:** Provides visual comparisons of drug safety profiles using Matplotlib.
- **Excel Integration:** Loads clinical data directly from external Excel files using Pandas.

## Technologies Used
- Python
- Pandas (Data Manipulation)
- Matplotlib (Data Visualization)
- Google Colab & GitHub
